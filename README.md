# 📊 Excel Files Merger

A Python automation tool that merges multiple Excel files into one professional report — with just one click!

Built with **Python**, **Pandas**, **openpyxl**, and **Tkinter**.

---

## 🖥️ GUI Preview

![GUI](screenshots/gui.png)

---

## ✨ Features

- Merges multiple Excel files from a folder automatically
- Adds **Source File** column to track data origin
- Creates a **Summary Sheet** with grouped sales data
- Professional formatting with colors, borders, and fonts
- Simple and clean GUI — no coding required to use
- Error handling for missing or incorrect files

---

## 📦 Requirements

Install dependencies using pip:

```bash
pip install pandas openpyxl
```

---

## 🚀 How to Use

1. Clone the repository:
```bash
git clone https://github.com/abdullahautomation/excel-files-merger.git
```

2. Navigate to the project folder:
```bash
cd excel-files-merger
```

3. Add your Excel files to the `excel_files` folder

4. Run the GUI:
```bash
python gui.py
```

5. Enter output file name
6. Click **Run** — done! ✅
